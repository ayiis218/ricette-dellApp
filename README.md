<!-- GETTING STARTED -->
## Getting Started

### Requirements
* [Node.js](https://nodejs.org/en/)
* [Postman](https://www.getpostman.com/)
* [Database](./blanja.sql)

### Installation

- Clone the Repo
```
git clone https://github.com/ayiis218/ricette-dellApp.git
```
- Go To Folder Repo
```
cd ricette-dellApp
```
- Install Module
```
npm install
```
